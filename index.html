2. Netflix Stock Price Analysis in SQL

This project leverages Netflix Stock Price Data from 2002 to 2022, sourced from Kaggle.com. The primary focus involves exploring the data within SQL Server by time series analysis method to gain profound insights into the longitudinal trends of stock prices. Additionally, I employ Tableau to visualize various facets of the data, enhancing the comprehensibility of the findings. An analysis summary report can be accessed via this link.
